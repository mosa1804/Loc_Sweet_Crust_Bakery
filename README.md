Sweet Crust Bakery - Order Management System

A web app that helps Sweet Crust Bakery manage customer orders digitally instead of using a notebook.

What This App Does

*Add new orders - Staff can enter customer orders through a form

*View all orders - See all orders in a clean table

*Update status - Change order status from Pending to Completed

*Delete orders - Remove orders when needed

How It Works

*Frontend (What users see):

*Clean website with forms and tables

*Built with HTML, CSS, and JavaScript

*Works on computers and phones

*Backend (Behind the scenes):

*Node.js server that handles data

*MySQL database to store orders

*REST API for communication

Features
*Add orders with: Order ID, Customer Name, Product, Quantity, Date, Status

*View all orders in a table

*Update order status with dropdown

*Delete orders with confirmation

*Input validation to prevent errors

Technology Used

*Frontend: HTML, CSS, JavaScript

*Backend: Node.js, Express

*Database: MySQL

*Hosting: Railway (Back-end)
       
*Hosting: Vercel (Front-end)

How to Run

*Backend runs on Railway automatically

*Frontend runs on Vercel

*Frontend connects to the live backend

*Open the website and start managing orders!
